# Advent of Code - 2025
You are a competitive programmer trying to solve Advent of code 2025. Follow the instructiosn strictly.
You can connect to chrome and browse the sites.

## Instructions
1. Create a folder for each day (format ./day_xx/ - xx day number in two digits).
2. Go to the site https://adventofcode.com/2025/day/<day-no> (day-no is the day number) using chrome.
3. Copy the puzzle input by following the steps
    a. Create the file ./day_xx/input.txt.
    a. Go to the site https://adventofcode.com/2025/day/<day-no>/input (day-no is the day number) using chrome and copy the content of the site to the clipboard .
    b. Use powershell -Command "Get-Clipboard | Out-File -FilePath 'day_xx\input.txt' -Encoding utf8" to copy the contents of the clipboard to day_xx\input.txt
4. Read the first part of the question, think and come up with the strategy. Write the thought process in ./day_xx/README.md file.
5. Write python code (./day_xx/part1.py)
6. Verify the script using the example in the puzzle.
7. After verifying that the output run the script on the puzzle input, get the answer.
8. Submit the answer. If you get the **wrong answer** pause and think again on the approach.
8. Then read the second part, think and come up with the strategy. Update the ./day_xx/README.md file.
9. Write python code (./day_xx/part2.py)
10. Verify the script using the example in the puzzle.
11. Run the script on the puzzle input, get the answer.
12. Submit the answer for part 2. If you get the **wrong answer** pause and think again on the approach.
